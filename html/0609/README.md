## tailwind css

- 如果使用 cdn, 加入 script 後就會內建 reset css ，避免瀏覽器預設樣式不同，他先將所有的樣式去掉
- 在根目錄放 tailwind.config.js 才會讓 extension 啟動
- tailwind css 沒有先後順序的問題
- 顏色後面 /數值，可以調透明度，如去 bg-black/30
- p-4 << 數值^4，習慣用 4 的倍數，像這邊就是 16px
